❔ Com aconsegueixo un token?

1. A GitHub, entra al teu avatar (dalt a la dreta) → Settings.
2. Baixa fins Developer settings (al final del menú esquerre).
3. Personal access tokens → Fine-grained tokens → Generate new token.
4. A "Repository access", tria Only select repositories i selecciona el teu repositori del dashboard.
5. A "Permissions → Repository permissions", busca Contents i posa'l en Read and write.
6. Genera el token i copia'l ara — GitHub només el mostra un cop.
7. Enganxa'l al camp de dalt i pulsa "Desa el token".
